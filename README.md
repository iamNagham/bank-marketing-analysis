# Bank Marketing Data Analysis

An exploratory data analysis project examining bank marketing campaign data to understand customer engagement patterns and the factors influencing campaign outcomes.

## Features

* Explore customer demographic and financial attributes
* Visualize relationships between customer profiles and campaign outcomes
* Summarize key patterns in campaign engagement

## Techniques Used

* Exploratory Data Analysis (EDA)
* Statistical Analysis
* Data Visualization

## Tools

* R
* ggplot2
* dplyr

## How it works

The script loads the bank marketing dataset and processes customer attributes such as age, job, marital status, and account balance. Statistical methods are applied to examine relationships between these attributes and campaign outcomes, while ggplot2 visualizations highlight patterns in customer engagement to help interpret which factors are associated with a successful campaign response.
